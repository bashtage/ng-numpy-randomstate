randomstate.prng.xorshift1024.seed¶

randomstate.prng.xorshift1024.seed(seed=None)¶

Seed the generator.

This method is called when RandomState is initialized. It can be called again to re-seed the generator. For details, see RandomState.

Parameters:

seed (int, optional) – Seed for RandomState.
Raises:
  • ValueError – If seed values are out of range for the PRNG.
  • TypeError – If seed values are not integers.

Notes

Acceptable range for seed depends on specifics of PRNG. See class documentation for details.

Seeds are hashed to produce the required number of bits.
