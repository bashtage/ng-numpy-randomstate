randomstate.prng.xorshift128.random_raw¶

randomstate.prng.xorshift128.random_raw(self, size=None)¶

Return randoms as generated by the underlying PRNG

Parameters:size (int or tuple of ints, optional) – Output shape. If the given shape is, e.g., (m, n, k), then m * n * k samples are drawn. Default is None, in which case a single value is returned.
Returns:out – Drawn samples.
Return type:uint or ndarray

Notes

This method directly exposes the the raw underlying pseudo-random number generator. All values are returned as unsigned 64-bit values irrespective of the number of bits produced by the PRNG.

See the class docstring for the number of bits returned.
